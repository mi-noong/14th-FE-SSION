자기소개

안녕하세요. 저는 Product Manager로서 서비스를 기획하다가, 직접 사용자 경험을 구현하고 싶어 프론트엔드 개발로 전향한 신진호입니다. 기획을 하면서 “좋은 아이디어도 결국 구현되지 않으면 사용자에게 닿지 않는다”는 한계를 느꼈고, 그 경험이 개발을 시작하게 된 계기가 되었습니다. 현재는 HTML, CSS, JavaScript를 기반으로 사용자에게 직관적이고 자연스러운 인터페이스를 만드는 데 집중하고 있습니다. 특히 기획 경험을 바탕으로 사용자 흐름을 고려한 UI 설계와, 실제 사용성을 개선하는 데 강점을 가지고 있습니다. 앞으로는 기획과 개발을 모두 이해하는 프론트엔드 개발자로서, 단순한 기능 구현을 넘어 사용자 경험 자체를 설계할 수 있는 개발자가 되고 싶습니다.

관심 기술

  • UI/UX 기획 및 사용자 흐름 설계
  • HTML, CSS, JavaScript 기반 웹 서비스 구조 이해
  • Figma를 활용한 화면 설계와 프로토타이핑
  • 데이터 기반 문제 정의와 서비스 개선
